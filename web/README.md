# `@kestrai/web` — Phase 2 stub

The Kestrai web GUI is **not** part of the v1 release. This directory exists so the monorepo's pnpm workspace already knows where the package will live; it ships intentionally empty until Phase 2 (Next.js 15 + RSC + Tailwind + shadcn/ui + TanStack Query + React Flow).

In Phase 0 and Phase 1, the supported UI surfaces are:
- The CLI (`kestrai`)
- The TUI (lands in Phase 1)
